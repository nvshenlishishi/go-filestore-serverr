## 8-2 阿里云对象存储OSS特点
### OSS的特点
- 可靠性 服务可用性，数据持久性
- 安全行 资源隔离存储，访问鉴权
- 易用性 标准restful风格api,多种语言SDK接入
- 处理能力 海量规模，图片处理，音视频转码

## 8-3 阿里云对象存储OSS专业术语
```
bucket          存储空间、桶
object          对象或者文件
endpoint        oss访问域名
region          区域或者数据中心
accesskey       资源访问密钥
object meta     文件元信息
data            文件数据
key             文件名
acl(Access Control list) 存储空间或者文件的权限
```
